## Matrices of the Same Map

If a linear operator <M math="T: V \to V" /> is represented in basis <M math="\mathcal{B}" /> by matrix <M math="A" />, and in basis <M math="\mathcal{B}'" /> by matrix <M math="A'" />, then (from the change-of-basis formula):

<BM math="A' = P^{-1} A P," />

where <M math="P" /> is the transition matrix. Two matrices related this way are called **similar** — they represent the same linear map in different bases.

---

## Definition

<DefinitionBox number={25} title="Similar Matrices">
Two matrices <M math="A, B \in M_{n\times n}(\mathbb{F})" /> are **similar** (written <M math="A \sim B" />) if there exists an invertible matrix <M math="P \in M_{n\times n}(\mathbb{F})" /> such that

<BM math="B = P^{-1} A P." />

Similarity is an **equivalence relation** on <M math="M_{n\times n}(\mathbb{F})" />:
- *Reflexive*: <M math="A = I^{-1}AI \sim A" />.
- *Symmetric*: <M math="B = P^{-1}AP \Rightarrow A = PBP^{-1} = (P^{-1})^{-1}B(P^{-1})" />.
- *Transitive*: <M math="B = P^{-1}AP" /> and <M math="C = Q^{-1}BQ \Rightarrow C = (PQ)^{-1}A(PQ)" />.
</DefinitionBox>

## Similarity Invariants

<TheoremBox
  number={31}
  title="Properties Preserved Under Similarity"
  proofSketch={
    <>
      Let <M math="B = P^{-1}AP" />.
      <br />
      <strong>Char poly:</strong> <M math="\det(\lambda I - B) = \det(\lambda I - P^{-1}AP) = \det(P^{-1}(\lambda I - A)P) = \det(P^{-1})\det(\lambda I - A)\det(P) = \det(\lambda I - A)" />.
      <br />
      <strong>Trace:</strong> <M math="\operatorname{tr}(P^{-1}AP) = \operatorname{tr}(A \cdot PP^{-1}) = \operatorname{tr}(A)" /> (cyclic property). ∎
    </>
  }
>
  If <M math="A \sim B" /> then:

  | Property | Value |
  |----------|-------|
  | Characteristic polynomial | <M math="p_A = p_B" /> |
  | Eigenvalues (with multiplicity) | Same |
  | Trace | <M math="\operatorname{tr}(A) = \operatorname{tr}(B)" /> |
  | Determinant | <M math="\det(A) = \det(B)" /> |
  | Rank | <M math="\operatorname{rank}(A) = \operatorname{rank}(B)" /> |
  | Nullity | <M math="\operatorname{nullity}(A) = \operatorname{nullity}(B)" /> |

  **Important:** The converse is **false** — matrices can share all these properties without being similar (see Example 4).
</TheoremBox>

<SummaryBox title="Key Takeaways">
- Similar matrices represent the **same linear operator** in different bases: <M math="B = P^{-1}AP" />.
- Similarity preserves: characteristic polynomial, eigenvalues, trace, determinant, rank, nullity.
- These are called **similarity invariants** — they describe intrinsic properties of the linear map.
- The converse fails: equal invariants do not guarantee similarity.
- The **Jordan Normal Form** (Section VI) gives a complete classification up to similarity.
</SummaryBox>
