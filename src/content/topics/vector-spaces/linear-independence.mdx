## Big Idea

A set of vectors is **linearly independent** if none of them is a "redundant" combination of the others — the only way a linear combination of them equals zero is if all the coefficients are zero. Linear independence is the key property that makes a set efficient for describing a vector space.

---

## Formal Definition

<DefinitionBox number={8} title="Linear Independence and Dependence">
Let $V$ be a vector space over $\mathbb{F}$ and let $S = \{\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_k\} \subseteq V$.

$S$ is **linearly independent (LI)** if the only solution to:
$$\alpha_1 \mathbf{v}_1 + \alpha_2 \mathbf{v}_2 + \cdots + \alpha_k \mathbf{v}_k = \mathbf{0}$$
is $\alpha_1 = \alpha_2 = \cdots = \alpha_k = 0$ (the **trivial solution**).

$S$ is **linearly dependent (LD)** if there exist scalars $\alpha_1, \ldots, \alpha_k \in \mathbb{F}$, **not all zero**, such that:
$$\alpha_1 \mathbf{v}_1 + \alpha_2 \mathbf{v}_2 + \cdots + \alpha_k \mathbf{v}_k = \mathbf{0}$$

**Conventions:**
- The empty set $\varnothing$ is linearly independent (vacuously).
- An infinite set $S$ is LI if every *finite* subset of $S$ is LI.
</DefinitionBox>

---

## Equivalent Characterisation

<TheoremBox number={7} title="LD ↔ One Vector is a Combination of the Others" proofSketch={<>(⇒) If Σαᵢvᵢ = 0 with some αⱼ ≠ 0, then vⱼ = -(αⱼ)⁻¹ Σᵢ≠ⱼ αᵢvᵢ. (⇐) If vⱼ = Σᵢ≠ⱼ βᵢvᵢ, then Σᵢ≠ⱼ βᵢvᵢ + (-1)vⱼ = 0, a non-trivial relation.</>}>
A finite set $\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$ is linearly dependent if and only if at least one vector $\mathbf{v}_j$ can be written as a linear combination of the others:
$$\mathbf{v}_j = \sum_{i \neq j} \alpha_i \mathbf{v}_i$$

Equivalently: a set is LI if and only if no vector in it is in the span of the remaining vectors.
</TheoremBox>

## Practical Test: Matrix Method

For vectors in $\mathbb{F}^n$, linear independence can be tested by row-reducing the matrix formed by placing the vectors as **columns**:

<BM math="\text{Columns } \mathbf{v}_1, \ldots, \mathbf{v}_k \text{ LI} \iff \text{rank}([\mathbf{v}_1 \mid \cdots \mid \mathbf{v}_k]) = k" />

> If $k > n$, any $k$ vectors in $\mathbb{F}^n$ are automatically LD (more vectors than dimensions).

## Key Observations

<TheoremBox number={8} title="Supersets of LD Sets are LD; Subsets of LI Sets are LI" proofSketch={<>If S is LD, Σαᵢvᵢ = 0 with not all αᵢ = 0. The same relation holds in any superset T ⊇ S (just set the new coefficients to 0). Contrapositive: if T is LI, every subset is LI.</>}>
- If $S$ is linearly dependent and $T \supseteq S$, then $T$ is linearly dependent.
- Equivalently: every subset of a linearly independent set is linearly independent.
- If $S$ is linearly independent and $\mathbf{v} \notin \operatorname{span}(S)$, then $S \cup \{\mathbf{v}\}$ is linearly independent.
</TheoremBox>

<SummaryBox title="Key Takeaways: Linear Independence">
- LI means: the only way to get $\mathbf{0}$ as a combination is with all-zero coefficients.
- LD means: some vector is a linear combination of the others — it's "redundant."
- Any set containing $\mathbf{0}$ is LD.
- In $\mathbb{F}^n$: test LI by column-reducing. More than $n$ vectors are always LD.
- LI is "hereditary downward": any subset of an LI set is LI.
- LD is "hereditary upward": any superset of a LD set is LD.
- LI + spanning = basis (coming next lesson).
</SummaryBox>
