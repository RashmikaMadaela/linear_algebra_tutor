## Section Overview

Vector spaces are the central object of linear algebra.
A vector space is a set $V$ over a field $F$ where you can add elements and scale them by scalars — and these operations satisfy eight axioms.
The richness of the theory comes from how abstract this is: vectors can be arrows in $\mathbb{R}^3$, polynomials, matrices, continuous functions, or sequences.

### The Six Key Ideas

1. **Vector Space** — the eight axioms and first examples (Def 4)
2. **Subspaces** — when is a subset itself a vector space? (Def 5, the subspace criterion Thm 5)
3. **Span** — the smallest subspace containing a given set (Def 6–7)
4. **Linear Independence** — when no vector is "redundant" in a set (Def 8–9)
5. **Basis** — a linearly independent spanning set; the "coordinate grid" (Def 10–12)
6. **Dimension** — the size of any basis; invariant of the space (Def 13, Thm 15)
