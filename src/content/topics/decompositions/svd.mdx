## Big Idea

Diagonalization $A = PDP^{-1}$ only works for square diagonalizable matrices.
**Singular Value Decomposition (SVD)** generalizes diagonalization to **every** matrix — any shape, any rank.
Every $m \times n$ matrix $A$ factors as $A = U\Sigma V^H$ where $U$ and $V$ are unitary and $\Sigma$ is "diagonal" (with nonneg. real entries).
The SVD is the most important matrix factorization in applied mathematics and data science.

## Intuition

Any linear map $T: \mathbb{R}^n \to \mathbb{R}^m$ (represented by $A$) can be decomposed into three steps:

1. **Rotate** the input space by $V^H$ — find the "right" input directions
2. **Scale** each direction by a singular value $\sigma_i$ (and possibly reduce/increase dimension via $\Sigma$)
3. **Rotate** the output space by $U$ — arrange the output in the "right" directions

The singular values $\sigma_1 \geq \sigma_2 \geq \cdots \geq 0$ measure how much $A$ stretches each direction. The right singular vectors $\mathbf{v}_i$ (columns of $V$) are the input directions; the left singular vectors $\mathbf{u}_i$ (columns of $U$) are the output directions.

## Preparatory Theorem

<TheoremBox number={40} title="Properties of $A^H A$ and $A A^H$" proofSketch={<>Hermitian: <M math="(A^H A)^H = A^H(A^H)^H = A^H A" />. PSD: <M math="\mathbf{x}^H A^H A\mathbf{x} = \|A\mathbf{x}\|^2 \geq 0" />. Same nonzero eigenvalues: if <M math="A^H A\mathbf{v} = \lambda\mathbf{v}" /> with <M math="\lambda > 0" />, then <M math="AA^H(A\mathbf{v}) = \lambda(A\mathbf{v})" /> and <M math="A\mathbf{v} \neq \mathbf{0}" />.</>}>

For any $A \in \mathbb{C}^{m \times n}$:

1. $A^H A \in \mathbb{C}^{n\times n}$ and $A A^H \in \mathbb{C}^{m \times m}$ are both **Hermitian and positive semidefinite**.
2. $A^H A$ and $A A^H$ have the **same nonzero eigenvalues** (with the same multiplicities).
3. $\operatorname{rank}(A^H A) = \operatorname{rank}(A A^H) = \operatorname{rank}(A)$.
4. All eigenvalues of $A^H A$ are **real and nonneg**: $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_n \geq 0$.
</TheoremBox>

## Formal Definition

<DefinitionBox number={34} title="Singular Values and Singular Vectors">
Let $A \in \mathbb{C}^{m \times n}$ with $\operatorname{rank}(A) = r$.
Let $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_r > 0 = \lambda_{r+1} = \cdots = \lambda_{\min(m,n)}$ be the eigenvalues of $A^H A$ (equivalently $AA^H$, for the nonzero ones).

The **singular values** of $A$ are:

<BM math="\sigma_i = \sqrt{\lambda_i} \geq 0, \quad i = 1, \ldots, \min(m,n)" />

with $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > \sigma_{r+1} = \cdots = 0$.

The **right singular vectors** $\mathbf{v}_1, \ldots, \mathbf{v}_n$ are the orthonormal eigenvectors of $A^H A$:
<BM math="A^H A \mathbf{v}_i = \sigma_i^2 \mathbf{v}_i" />

The **left singular vectors** $\mathbf{u}_1, \ldots, \mathbf{u}_m$ are defined by:
<BM math="\mathbf{u}_i = \frac{1}{\sigma_i} A \mathbf{v}_i \quad (i \leq r), \qquad \text{extended to orthonormal basis of } \mathbb{C}^m \text{ for } i > r" />

The relation $A\mathbf{v}_i = \sigma_i \mathbf{u}_i$ holds for $i = 1, \ldots, r$.
</DefinitionBox>

## SVD Theorem

<TheoremBox number={41} title="Singular Value Decomposition (SVD)" proofSketch={<>Diagonalize <M math="A^H A = V\Lambda V^H" /> (by spectral theorem for Hermitian matrices). Define <M math="\mathbf{u}_i = A\mathbf{v}_i/\sigma_i" /> for <M math="i \leq r" /> and extend to orthonormal basis of <M math="\mathbb{C}^m" />. Verify <M math="A = U\Sigma V^H" /> by checking both sides agree on all basis vectors.</>}>

Every $A \in \mathbb{C}^{m \times n}$ can be written as

<BM math="A = U \Sigma V^H" />

where:
- $U \in \mathbb{C}^{m\times m}$ is **unitary** (columns = left singular vectors)
- $V \in \mathbb{C}^{n \times n}$ is **unitary** (columns = right singular vectors)
- $\Sigma \in \mathbb{R}^{m \times n}$ is **"diagonal"**: $\Sigma_{ii} = \sigma_i \geq 0$, all other entries $0$

The singular values $\sigma_1 \geq \sigma_2 \geq \cdots \geq 0$ are unique. The SVD always exists.

**Rank:** $\operatorname{rank}(A) = r$ = number of nonzero singular values.

**Compact SVD:** $A = \sum_{i=1}^r \sigma_i \mathbf{u}_i \mathbf{v}_i^H$ (outer product / dyadic expansion).
</TheoremBox>

## Summary

<SummaryBox title="SVD — Key Facts">

**Every** matrix $A \in \mathbb{C}^{m \times n}$ has $A = U\Sigma V^H$.

| Quantity | Meaning |
|----------|---------|
| $\sigma_1$ | $\|A\|_2$ (spectral/operator norm) |
| $\sigma_r$ (smallest nonzero) | Distance to nearest rank-$(r-1)$ matrix |
| $\sigma_1/\sigma_r$ | Condition number $\kappa(A)$ |
| $\sum \sigma_i^2$ | $\|A\|_F^2$ (Frobenius norm squared) |
| Number of nonzero $\sigma_i$ | $\operatorname{rank}(A)$ |

**Key formula:** $A = \sum_{i=1}^r \sigma_i \mathbf{u}_i \mathbf{v}_i^H$ (outer product expansion — most important form).

**Applications:** PCA, image compression, least squares (pseudoinverse), recommender systems, noise reduction.
</SummaryBox>
