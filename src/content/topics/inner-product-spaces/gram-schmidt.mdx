## Big Idea

We know orthonormal bases are ideal for computation.
But what if we have a linearly independent set that isn't orthogonal?
The **Gram-Schmidt process** is a systematic algorithm that converts any linearly independent set into an orthonormal one **spanning the same subspace**.
It works by repeatedly subtracting the "already-covered" component from each new vector.

## Intuition

Start with two non-orthogonal vectors $\mathbf{u}_1, \mathbf{u}_2$.
- Keep $\mathbf{u}_1$ as the first basis direction.
- Subtract from $\mathbf{u}_2$ its projection onto $\mathbf{u}_1$. The remainder is orthogonal to $\mathbf{u}_1$.
- Normalize both.

For three vectors, after handling $\mathbf{u}_1$ and $\mathbf{u}_2$, subtract from $\mathbf{u}_3$ its projections onto **both** $e_1$ and $e_2$ — the remainder is orthogonal to the entire plane spanned so far.

The projection of $\mathbf{u}$ onto a unit vector $\mathbf{e}$ is $\langle\mathbf{u},\mathbf{e}\rangle\,\mathbf{e}$.
The "leftover" after removing this projection is $\mathbf{u} - \langle\mathbf{u},\mathbf{e}\rangle\,\mathbf{e}$.

## The Algorithm

<TheoremBox number={21} title="Gram-Schmidt Orthogonalization" proofSketch={<>Each <M math="v_k" /> is constructed by subtracting its projections onto all previous <M math="e_1,\ldots,e_{k-1}" />, so <M math="\langle v_k, e_j\rangle=0" /> for <M math="j < k" /> by design. Since <M math="\{u_1,\ldots,u_n\}" /> is LI, no <M math="v_k" /> can be zero. Normalizing gives <M math="e_k=v_k/\|v_k\|" />. Induction shows <M math="\operatorname{span}\{e_1,\ldots,e_k\}=\operatorname{span}\{u_1,\ldots,u_k\}" /> at each step.</>}>

Given a linearly independent set $\{u_1, u_2, \ldots, u_n\}$ in an inner product space $V$, the following process produces an **orthonormal set** $\{e_1, e_2, \ldots, e_n\}$ with the same span.

**Step 1:**
<BM math="v_1 = u_1, \qquad e_1 = \frac{v_1}{\|v_1\|}" />

**Step $k$ (for $k = 2, 3, \ldots, n$):**
<BM math="v_k = u_k - \sum_{j=1}^{k-1} \langle u_k,\, e_j\rangle\, e_j" />
<BM math="e_k = \frac{v_k}{\|v_k\|}" />

At each stage, $\operatorname{span}\{e_1,\ldots,e_k\} = \operatorname{span}\{u_1,\ldots,u_k\}$.
</TheoremBox>

**Key formula:** The projection of $\mathbf{u}$ onto $e_j$ is $\langle\mathbf{u}, e_j\rangle\, e_j$.
So $v_k$ is $u_k$ with all projections onto the previously computed orthonormal vectors removed.

## Common Mistakes

⚠ **Must start with LI set:** If the input vectors are linearly dependent, some $v_k$ will be zero and the algorithm breaks. Always check linear independence first.

⚠ **Project onto unit vectors:** The formula $v_k = u_k - \sum_j \langle u_k,e_j\rangle e_j$ uses the normalized $e_j$, not the intermediate $v_j$.

⚠ **Same span:** Gram-Schmidt produces an orthonormal basis for $\operatorname{span}\{u_1,\ldots,u_n\}$ — not necessarily for all of $V$ if $\{u_i\}$ doesn't span $V$.

## Summary

<SummaryBox title="Gram-Schmidt — Key Facts">

**Input:** Linearly independent set $\{u_1,\ldots,u_n\}$.

**Output:** Orthonormal set $\{e_1,\ldots,e_n\}$ with $\operatorname{span}\{e_1,\ldots,e_k\} = \operatorname{span}\{u_1,\ldots,u_k\}$ for each $k$.

**Iteration:** $v_k = u_k - \displaystyle\sum_{j=1}^{k-1}\langle u_k,e_j\rangle e_j$, then $e_k = v_k/\|v_k\|$.

**Application:** Proves every finite-dimensional inner product space has an orthonormal basis.

**Next steps:** Gram-Schmidt applied to $\{1, x, x^2,\ldots\}$ on $[-1,1]$ gives Legendre polynomials (next lesson), and provides the foundation for QR decomposition (Section VI).
</SummaryBox>
