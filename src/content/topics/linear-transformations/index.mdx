## Section Overview

Linear transformations are the *maps* between vector spaces — functions that respect the algebraic structure.
Every matrix multiplication $A\mathbf{x}$ is a linear transformation; conversely, every linear transformation between finite-dimensional spaces can be represented as a matrix (once bases are chosen).
This section bridges abstract linear maps and concrete matrix computations.

### The Five Key Ideas

1. **Definition** — what makes a map "linear" ($T(\alpha\mathbf{u}+\beta\mathbf{v}) = \alpha T\mathbf{u} + \beta T\mathbf{v}$)
2. **Kernel & Range** — the "collapsed" subspace and the "image" subspace
3. **Rank-Nullity Theorem** — $\text{null}(T) + \text{rank}(T) = \dim(V)$
4. **Matrix of a Linear Transformation** — coordinates and change of basis
5. **Matrix Spaces & RREF** — row/column space, rank, solutions to $A\mathbf{x} = \mathbf{b}$

### Theorems in this section
- **Thm 23** — $T(\mathbf{0}) = \mathbf{0}$, $T(-\mathbf{u}) = -T(\mathbf{u})$
- **Thm 24** — $\ker T$ and $\text{ran}\, T$ are subspaces; nullity and rank are well-defined
- **Thm 25 (Rank-Nullity)** — $\text{null}(T) + \text{rank}(T) = \dim V$
- **Thm 26** — Change of basis: $A' = Q^{-1}AQ$
- **Thm 27** — Row rank = column rank = rank
