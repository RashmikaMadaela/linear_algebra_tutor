## Big Idea

Every linear map between finite-dimensional vector spaces can be encoded as a matrix — once bases are chosen. The matrix representation turns abstract linear maps into concrete matrix multiplication, connecting the abstract theory to computational practice.

---

## Coordinate Vectors

<DefinitionBox number={18} title="Coordinate Vector">
Let $V$ be an $n$-dimensional vector space with **ordered basis** $\mathcal{B} = (\mathbf{b}_1, \ldots, \mathbf{b}_n)$. Since every $\mathbf{v} \in V$ has a unique representation $\mathbf{v} = \alpha_1\mathbf{b}_1 + \cdots + \alpha_n\mathbf{b}_n$, define the **coordinate vector** of $\mathbf{v}$ with respect to $\mathcal{B}$:

$$[\mathbf{v}]_\mathcal{B} = \begin{pmatrix} \alpha_1 \\ \vdots \\ \alpha_n \end{pmatrix} \in \mathbb{F}^n$$

The map $\phi_\mathcal{B}: V \to \mathbb{F}^n$ defined by $\phi_\mathcal{B}(\mathbf{v}) = [\mathbf{v}]_\mathcal{B}$ is a linear **isomorphism** (coordinate map).
</DefinitionBox>

---

## Matrix of a Linear Map

<DefinitionBox number={19} title="Matrix Representation [T]_{C,B}">
Let $T: V \to W$ be linear, $\mathcal{B} = (\mathbf{b}_1, \ldots, \mathbf{b}_n)$ an ordered basis for $V$, and $\mathcal{C} = (\mathbf{c}_1, \ldots, \mathbf{c}_m)$ an ordered basis for $W$.

The **matrix representation** of $T$ with respect to $\mathcal{B}$ (domain) and $\mathcal{C}$ (codomain) is the $m \times n$ matrix:

$$[T]_{\mathcal{C}, \mathcal{B}} = \Big([T(\mathbf{b}_1)]_\mathcal{C} \;\Big|\; [T(\mathbf{b}_2)]_\mathcal{C} \;\Big|\; \cdots \;\Big|\; [T(\mathbf{b}_n)]_\mathcal{C}\Big)$$

The $j$-th column of $[T]_{\mathcal{C},\mathcal{B}}$ is the coordinate vector of $T(\mathbf{b}_j)$ in the basis $\mathcal{C}$.

**Key property:**
$$[T(\mathbf{v})]_\mathcal{C} = [T]_{\mathcal{C},\mathcal{B}}\, [\mathbf{v}]_\mathcal{B}$$
</DefinitionBox>

---

## Matrix Representation Respects Composition

<TheoremBox number={23} title="Composition → Matrix Multiplication" proofSketch={<>For any v∈U: the matrix equation [S∘T]_(D,B) = [S]_(D,C) · [T]_(C,B) follows by evaluating [S∘T(v)] = [S]·[T(v)] = [S]·[T]·[v] in the respective bases. Since this holds for all v, matrix multiplication correctly represents composition of linear maps.</>}>
Let $T: U \to V$ and $S: V \to W$ be linear, with ordered bases $\mathcal{B}$ for $U$, $\mathcal{C}$ for $V$, $\mathcal{D}$ for $W$. Then:

$$[S \circ T]_{\mathcal{D}, \mathcal{B}} = [S]_{\mathcal{D}, \mathcal{C}}\, [T]_{\mathcal{C}, \mathcal{B}}$$

**Corollary:** Matrix multiplication is defined precisely so that it corresponds to composition of linear maps. This is the reason matrix multiplication is defined the way it is.
</TheoremBox>

## The Commutative Diagram

For $T: V \to W$, the matrix representation satisfies:

<BM math="\phi_\mathcal{C} \circ T = [T]_{\mathcal{C},\mathcal{B}} \circ\, \phi_\mathcal{B}" />

That is, the following diagram commutes:

$$V \xrightarrow{\;T\;} W$$
$$\downarrow\phi_\mathcal{B} \qquad\quad \downarrow\phi_\mathcal{C}$$
$$\mathbb{F}^n \xrightarrow{[T]_{\mathcal{C},\mathcal{B}}} \mathbb{F}^m$$

To compute $T(\mathbf{v})$:
1. Compute $[\mathbf{v}]_\mathcal{B}$ (coordinates in $\mathcal{B}$)
2. Multiply by $[T]_{\mathcal{C},\mathcal{B}}$
3. Convert back: $T(\mathbf{v}) = \phi_\mathcal{C}^{-1}\!\left([T]_{\mathcal{C},\mathcal{B}}[\mathbf{v}]_\mathcal{B}\right)$

<SummaryBox title="Key Takeaways: Matrix Representations">
- Fix ordered bases $\mathcal{B}$ and $\mathcal{C}$: $[T]_{\mathcal{C},\mathcal{B}}$ has $j$-th column $= [T(\mathbf{b}_j)]_\mathcal{C}$.
- Key formula: $[T(\mathbf{v})]_\mathcal{C} = [T]_{\mathcal{C},\mathcal{B}}[\mathbf{v}]_\mathcal{B}$.
- Composition corresponds to matrix multiplication: $[S \circ T] = [S][T]$.
- Different basis choices give different matrices for the **same** linear map.
- Choosing an eigenbasis (if it exists) gives the diagonal matrix — motivation for eigenvalue theory.
</SummaryBox>
