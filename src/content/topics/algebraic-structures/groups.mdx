## Binary Operations

Before defining a group, we need the concept of a binary operation — a rule that combines two elements of a set to produce another element of the same set.

<DefinitionBox number={1} title="Binary Operation">
A **binary operation** on a non-empty set $G$ is a mapping

$$\ast : G \times G \to G$$

that assigns to each ordered pair $(a, b) \in G \times G$ a unique element $a \ast b \in G$.

The key property is **closure**: for all $a, b \in G$, the result $a \ast b$ must also belong to $G$.
</DefinitionBox>

**Examples of binary operations:**
- Addition $+$ on $\mathbb{Z}$: for any two integers, their sum is also an integer. ✓
- Division $\div$ on $\mathbb{Z}$: $3 \div 2 \notin \mathbb{Z}$. ✗ (not closed)
- Matrix multiplication on $M_{n \times n}(\mathbb{R})$: closed. ✓

## Groups

<DefinitionBox number={2} title="Group">
A **group** is a non-empty set $G$ together with a binary operation $\ast$ on $G$ such that the following axioms hold:

1. **Associativity**: For all $a, b, c \in G$, $\;(a \ast b) \ast c = a \ast (b \ast c)$.
2. **Identity**: There exists an element $e \in G$ such that $e \ast a = a \ast e = a$ for all $a \in G$.
3. **Inverses**: For each $a \in G$, there exists $a^{-1} \in G$ such that $a \ast a^{-1} = a^{-1} \ast a = e$.

We denote this group $(G, \ast)$, or simply $G$ when the operation is clear.
</DefinitionBox>

<DefinitionBox number={3} title="Abelian Group">
A group $(G, \ast)$ is called **abelian** (or **commutative**) if it additionally satisfies:

4. **Commutativity**: For all $a, b \in G$, $\; a \ast b = b \ast a$.
</DefinitionBox>

## Fundamental Theorems

<TheoremBox number={1} title="Uniqueness of the Identity" proofSketch={<>Suppose <M math="e" /> and <M math="e'" /> are both identities. Then <M math="e = e \ast e' = e'" />, since <M math="e'" /> acts as an identity on <M math="e" />, and <M math="e" /> acts as an identity on <M math="e'" />.</>}>
The identity element of a group $(G, \ast)$ is **unique**.
</TheoremBox>

<TheoremBox number={2} title="Uniqueness of Inverses" proofSketch={<>Suppose <M math="b" /> and <M math="c" /> are both inverses of <M math="a" />. Then: <M math="b = b \ast e = b \ast (a \ast c) = (b \ast a) \ast c = e \ast c = c" />.</>}>
For each element $a$ in a group $(G, \ast)$, its inverse is **unique**.
</TheoremBox>

<TheoremBox number={3} title="Cancellation Laws">
In a group $(G, \ast)$, for all $a, b, c \in G$:

- **Left cancellation**: $a \ast b = a \ast c \implies b = c$
- **Right cancellation**: $b \ast a = c \ast a \implies b = c$
</TheoremBox>

<TheoremBox number={4} title="Socks-Shoes Property" proofSketch={<><M math="(a \ast b) \ast (b^{-1} \ast a^{-1}) = a \ast (b \ast b^{-1}) \ast a^{-1} = a \ast e \ast a^{-1} = a \ast a^{-1} = e" />. So <M math="b^{-1} \ast a^{-1}" /> is the inverse of <M math="a \ast b" />.</>}>
For any $a, b \in G$:

$$(a \ast b)^{-1} = b^{-1} \ast a^{-1}$$
</TheoremBox>

## Order of a Group and Subgroups

<DefinitionBox number={4} title="Order of a Group">
The **order** of a group $G$, written $|G|$ or $\text{ord}(G)$, is the number of elements in $G$.

- If $|G| < \infty$, we say $G$ is a **finite group**.
- If $|G| = \infty$, we say $G$ is an **infinite group**.
</DefinitionBox>

<DefinitionBox number={5} title="Order of an Element">
The **order** of an element $a \in G$, written $\text{ord}(a)$, is the smallest positive integer $n$ such that $a^n = e$. If no such $n$ exists, we say $a$ has **infinite order**.
</DefinitionBox>

<SummaryBox title="Key Takeaways: Groups">
- A **group** $(G, \ast)$ satisfies four axioms: closure, associativity, identity, and inverses.
- An **abelian group** additionally satisfies commutativity.
- Key examples: $(\mathbb{Z}, +)$, $(\mathbb{Q} \setminus \{0\}, \cdot)$, $(\mathbb{Z}_n, +_n)$, $GL_n(\mathbb{R})$ (invertible matrices under multiplication).
- The identity and each inverse are unique.
- The socks-shoes property: $(ab)^{-1} = b^{-1}a^{-1}$.
</SummaryBox>
